# Architecture Decision Records

This directory stores the decision log for pdf_toolbox. Each ADR captures the
context, decision, and follow-up considerations for an architectural change.

## Index

| ADR | Title | Status | Date | Summary |
| --- | ----- | ------ | ---- | ------- |
| [0001](0001-pptx-provider-architecture.md) | PPTX Provider Architecture | Accepted | 2024-05-20 | Defines the provider model for PPTX rendering and the Microsoft Office provider constraints. |

When proposing a new architectural change, add a numbered ADR to this directory
and update the index above. Use the next sequential identifier and keep the
format consistent with the existing records.
